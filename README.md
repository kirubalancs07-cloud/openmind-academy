# OpenMind Academy — Free Aptitude & Reasoning Platform
OpenMind Academy is a 100% free, open-source, mobile-friendly educational platform designed to help students master Quantitative Aptitude, Logical Reasoning, and Communication English. 
Built with low-resource and underprivileged students in mind, it requires no wallets, no paywalls, and no account sign-ups.
## 🌟 Features
- **Syllabus Coverage**: 56 comprehensive topics across 17 learning tracks.
- **English Course**: Custom-built module with daily self-study roadmaps, tenses tables, polite expressions, and a 150+ day-to-day vocabulary dictionary.
- **840 Solved Examples**: Exactly 15 step-by-step solved examples per topic to reinforce learning without needing video streaming or high-speed internet.
- **Interactive Quizzes**: Exam-pattern MCQs with instant correctness feedback and solution panels.
- **Privacy First**: Progress, bookmarks, and quiz scores are stored 100% locally in the browser cache (`localStorage`). No user data is sent to external servers.
## Tech Stack
- **Frontend**: Semantic HTML5, CSS3 (Glassmorphism & animations), TailwindCSS (Utility layout helper).
- **Core Engine**: Vanilla Javascript (SPA routing, particles background simulation, local storage data syncing).
